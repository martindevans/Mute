# \*Mute

\*Mute is a Discord bot, intended for my own use in private servers.

## Supporting Packages

- [Autofocus](https://github.com/martindevans/Autofocus): Originally a wrapper for the [AUTOMATIC1111](https://github.com/automatic1111/stable-diffusion-webui) API for image generation. Adapted to also support [Stable Diffusion.cpp](https://github.com/leejet/stable-diffusion.cpp).
- [BalderHash](https://github.com/martindevans/BalderHash): Generates memorable strings from numbers. e.g. `pagmyr-hapnex` is easier to remember than `284490538`.
- [Mute.Anilist](https://github.com/martindevans/Mute.Anilist): Wrapper for the [Anilist API](https://docs.anilist.co/).
- [Mute.BraveSearch](https://github.com/martindevans/Mute.BraveSearch): Wrapper for the [Brave Search API](https://api-dashboard.search.brave.com/documentation/quickstart).
- [Serpent](https://github.com/martindevans/Serpent): Provides safe sandboxed Python execution within a WASM container.
